# Iniciando aplicação

Clone o repositório:
```
git clone git@github.com:JefersonViana/desafio-tecnico.git
```
No terminal, execute o comando:
```
cd desafio-tecnico
```
# Instale as dependências e execute os testes
No terminal, execute os comandos:

Usando yarn

```bash
yarn install
yarn test
```

Ou usando o NPM:

```bash
npm install
npm test
```
